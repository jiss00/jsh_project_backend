## 프로젝트 소개

이 프로젝트는 React와 Spring Boot를 사용하여 토익 점수에 따라 책을 추천하고, 리뷰 게시판을 운영하며, 책의 장바구니 및 구매 기능을 구현한 웹 애플리케이션입니다.

## 주요 기능

- 토익 점수에 따라 책을 추천
- 각 책에 대한 리뷰 게시판 운영
- 책의 장바구니 및 구매 기능 구현 (포트원 API를 이용한 결제 테스트)
